01 / 05
Retail Sales Forecasting — Rossmann Stores
MSA Course · Group Project

End-to-end sales forecasting pipeline on 1M+ rows of German pharmacy chain data. Engineered calendar, lag, and rolling-window features, then compared Linear Regression, Random Forest, and Gradient Boosting regressors using time-series cross-validation.

Python Random Forest Gradient Boosting Time-Series CV Feature Engineering Plotly
Key Result

Gradient Boosting outperformed Linear Regression baseline; log-transformed Sales target improved model stability. Lag features and rolling averages captured weekly seasonality patterns.

02 / 05
SMS Spam Detection — NLP Classification
MSA 8150 · Machine Learning for Analytics · HW1

Multi-model spam classifier trained on the UCI SMS Spam Collection. Applied TF-IDF vectorization and benchmarked four algorithms — Logistic Regression, Naïve Bayes, Decision Tree, and KNN — using precision, recall, and F1-score.

NLP TF-IDF Logistic Regression Naïve Bayes KNN Decision Tree
Key Result

Comparative analysis across 4 classifiers on 5,574 messages. EDA uncovered clear distinctions: spam messages are longer and lexically distinct from ham (e.g. "free", "win", "claim").

03 / 05
Building Energy Efficiency — Regression Modeling
MSA 8150 · Machine Learning for Analytics · HW2

Predicted building Heating Load from 8 architectural features (UCI Energy Efficiency dataset). Implemented and systematically compared Linear Regression, Polynomial Regression (degree=2), and Ridge Regression with GridSearchCV hyperparameter tuning.

Linear Regression Polynomial Regression Ridge Regression GridSearchCV StandardScaler KFold CV
Key Result

Ridge Regression on polynomial features achieved the best test R² and RMSE. Bias-variance trade-off curve showed degree-3+ caused overfitting; optimal alpha found via 5-fold cross-validation.

04 / 05

Sentiment classification pipeline on customer product reviews. Applied TF-IDF text vectorization, label encoding, and Logistic Regression to predict review sentiment at scale — including deduplication, null-handling, and structured evaluation.

NLP Sentiment Analysis TF-IDF Logistic Regression Seaborn pandas

Full pipeline from raw CSV to classification report — preprocessing, vectorization, training, and confusion matrix evaluation — structured for reproducibility with modular functions.
